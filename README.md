# Location-Maps-with-Python

# Overview
This repository contains location maps developed in Python for cartographic representation of municipalities, regions, and study areas. The maps are designed for geographic visualization, spatial reference, and support for environmental and territorial analyses.


# Objectives
- Create location maps for geographic and environmental applications
- Improve spatial visualization of study areas
- Apply Python tools for cartographic representation

# Data
  - Official shapefiles from public geographic databases such as that of the IBGE


# Tools and Libraries
  - Python
  - Matplotlib
  - Cartopy
  - Geopandas

    
# Outputs
  - Location map of a city
  - Location map of the brazilian regions
